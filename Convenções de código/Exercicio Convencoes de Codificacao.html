Exercício - Convenções de Codificação

Para resolução destas questões, utilize as classes contidas neste arquivo.
Considerando as seguintes convenções de codificação

  • Arquivo Java com terminação .java
  • Tamanho máximo de 2000 linhas
  • Importações apenas de classes necessárias, sem importações duplicadas e uso de *
  • Linhas com menos de 80 caracteres
  • Documentação de classe
  • Documentação de métodos com parâmetros, retornos e exceções
  • Blocos sempre delimitados por { }
  • Nome de pacote no padrão lower.case
  • Nome de classe no padrão CamelCase
  • Nome de atributo estático final no padrão UPPER_CASE
  • Nome de atributo estático no padrão camelCase
  • Nome de atributo de instância no padrão camelCase
  • Nome de método no padrão camelCase
  • Nome de variável local no padrão camelCase
  • Nome de parâmetro de método no padrão camelCase
  • Uso de constantes para valores mágicos
